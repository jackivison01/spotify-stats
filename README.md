# Spotify Stats
React app that allows you to check your live Spotify stats via the Spotify API

# Pages
* Account page - where you can see the following
    * Name
    * Profile Photo
    * Recently Played track
    * Top 3 artists
    * Top 3 songs
    * Top 3 albums
